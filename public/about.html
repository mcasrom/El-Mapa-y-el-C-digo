1. ¿Qué es BlogGPT?

BlogGPT es un sistema de blogging minimalista construido con Emacs y Org-mode. Diseñado para desarrolladores, escritores técnicos y entusiastas de Emacs que desean:

  • Mantener su contenido en formato de texto plano (Org-mode)
  • Aprovechar el poder de Emacs y su ecosistema
  • Tener control total sobre la generación y publicación de su blog
  • Mantener un flujo de trabajo sencillo y basado en git

3. Tecnologías Utilizadas

Este blog está construido con:

  • Emacs y Elisp
  • Org-mode para la redacción de contenido
  • HTML/CSS para la presentación
  • Git para el control de versiones y despliegue
