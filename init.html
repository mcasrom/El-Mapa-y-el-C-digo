1. Inicialización del Blog

Este blog está construido con Org-mode, una herramienta poderosa para gestionar contenido y publicarlo como un sitio web estático. Esta página sirve como punto de partida para entender cómo se configura y mantiene el blog.

1.1. Configuración

  • Herramientas: Emacs, Org-mode, y GitHub Pages.
  • Estructura: Posts en `~/org-blog/content/posts/` y páginas estáticas como esta.
  • Publicación: Automatizada con `org-publish`.

Explora los posts para ver ejemplos de contenido o visita la página de contacto para conectar.
